# Proyectos Nilus — Automatizaciones y Optimización de Procesos (Python)

Este repositorio reúne scripts y utilidades en Python orientados a optimizar tareas operativas de distintas áreas de una empresa (logística/operaciones, prevención, seguimiento, soporte interno, etc.).  
El objetivo es centralizar automatizaciones, estandarizar configuración y facilitar el mantenimiento.

---

## Contenidos principales

Actualmente el repo está organizado por áreas/temas. A modo orientativo:

```
.
├─ CANCELACIÓN-URL/        # Scripts para cancelar pedidos/procesos por lote
├─ Desvios operativos/     # Scripts para identificar/gestionar desvíos operativos
├─ Prevención/             # Scripts para validaciones, controles preventivos y saneo de datos
└─ README.md
```

Cada carpeta contiene uno o más scripts y, en algunos casos, módulos de apoyo (utils) y/o subcarpetas. Si una carpeta incluye su propio README, consúltalo para detalles específicos de ejecución.# Proyectos-Nilus
